# Request: 2026-02-18__consistency-check-multi-task-mode

## 要望の原文

- `check_consistency` を単一 task-id だけでなく複数 task を検査できるようにするタスクを起票する。

## 要望の整理

- `-TaskId` に加えて複数対象を扱う実行モードを追加する。
- CI / ローカルの両方で再利用しやすい実行インターフェースにする。
- 失敗集計をタスク単位で出力できるようにする。

## 成功条件（要望レベル）

1. 単一 task との後方互換を維持する。
2. 複数 task 一括検査が可能になる。
3. いずれか失敗時に終了コード 1 を返す。
